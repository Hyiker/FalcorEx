# Falcor Extended

This is an extended/patched version of [NVIDIA Falcor](https://github.com/NVIDIAGameWorks/Falcor), for testing real-time rendering techniques and algorithms.

## Changelog

- Bug fixes:
  - Double sided emissive material fix
  - Assimp bumped to v5.2.5 for better glTF support
  - ply file import fix using rply
- Additional features:
  - Added support for animated vertex data in glTF files
  - tiny-cuda-nn integration
